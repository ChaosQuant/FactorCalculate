# FactorCalculate

### /factor

该文件目录下保存的是因子计算的task文件， 以及每个task所依赖的因子计算文件。

目前包含的task：

- factor_growth  # 历史成长
- historical_values  # 价值
- factor_cash_flow  # 收益质量
- factor_constrain  # 收益质量
- factor_earning  # 收益质量
- factor_per_share_indicators  # 收益质量

### /client
程序执行入口, 文件目录下包含单类因子计算， 以及合并计算

### sumbit
分布式计算，提交入口

### init
初始化分布式计算redis数据库

### cluster_work
分布式引擎节点启动程序
