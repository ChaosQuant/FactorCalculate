# FactorCalculate

- /factor

该文件目录下保存的是因子计算的task文件， 以及每个task所依赖的因子计算文件。

目前包含的task：
    - factor_growth.py  # 历史成长
    - factor_per_share_indicators.py  # 规模，未使用
    - historical_values.py  # 价值
    - factor_cash_flow.py  # 收益质量
    - factor_constrain.py  # 收益质量
    - factor_earning.py  # 收益质量

- client.py

程序执行入口

- sumbit.py

分布式计算，提交入口
